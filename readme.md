# Paperless


# Objetivo

  - Se propone realizar una API que permita Crear y consultar tickets digitales

El proyecto esta enfocado a una solucion Web API integrado por los siguientes elementos:

  - Persistencia BD
  - Web API
  - Conversor de plantillas
